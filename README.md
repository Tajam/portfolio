## Development

```bash
# install dependencies
$ npm install

# serve with hot reload at 0.0.0.0:22222
$ npm run dev
```

For detailed explanation on how things work, check out [Nuxt.js docs](https://nuxtjs.org).

## Deploy (Window)

```bash
# this will build the SPA in a special way
$ npm run build

# toss it on Github
$ npm run deploy
```
